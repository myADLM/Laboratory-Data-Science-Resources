
# Contributing

We welcome resources that help laboratorians apply data science responsibly.

## What to submit
- Peer‑reviewed articles, high‑quality tutorials, courses, datasets, tools.
- Must include a 1–2 sentence summary + tags:
  - Level: Beginner / Intermediate / Advanced
  - Modality: Article / Course / Tool / Dataset
  - Lab Context: QC / Validation / LIS / Ops / Clinical ML

## How to submit
Open an Issue using "Suggest a resource" or a Pull Request adding one item under the correct section.

## Review
Maintainers review weekly for relevance, quality, and lab applicability. Broken links or vendor‑only promos are declined.
